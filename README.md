# Painel de Formulário

Projeto de painel de formulário com frontend e backend. Permite que o usuário digite nome, e-mail e telefone, valida os campos e envia os dados para um banco MySQL.

## Funcionalidades

- Formulário com campos: nome, e-mail e telefone  
- Validação de inputs no frontend  
- Envio dos dados para o banco MySQL  
- Backend em Node.js com Express

## Tecnologias

- Frontend: HTML, CSS, JavaScript, Tailwind  
- Backend: Node.js, Express  
- Banco de dados: MySQL

## Banco de dados

- Usuário: `root`  
- Senha: `1234`  
- O banco já está configurado no backend.
